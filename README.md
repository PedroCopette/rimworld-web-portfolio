# Portfólio RimWorld

Esse projeto é um site que eu fiz inspirado no jogo RimWorld.

A ideia foi criar uma página tipo apresentação do jogo, com informações, imagens e algumas interações usando HTML, CSS e um pouco de JavaScript.

---

## Sobre

O site mostra:

- Uma descrição do jogo  
- Galeria de imagens com carrossel  
- Trailer do jogo  
- Informações básicas (gênero, plataforma, etc)  
- Destaques do jogo  
- Um formulário de suporte  

---

## Tecnologias usadas

- HTML  
- CSS  
- JavaScript  

---

## O que tem no projeto

- Fundo animado com estrelas  
- Botão que mostra/esconde a parte "Sobre"  
- Carrossel de imagens automático  
- Formulário com mensagem de envio  
- Botão de voltar ao topo  
- Layout adaptado pra celular  

---

## Como rodar

Só abrir o arquivo `index.html` no navegador.
